После внесения всех изменений в базу данных командами, нам потребуется создать контроллеры, для этого идем в папку Controllers, клик левой кнопкой мыши - Add - Controllers
Добавляем все необходимые контроллеры, в конечном итоге у меня получилось вот так:

( screen ControllersAdd )

Теперь займемся настройкой этих контроллеров.

Для этого я написал запросы как показано ниже:

( screen - ControllerRequests )
